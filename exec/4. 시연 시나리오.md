# 🚗 스마트 주차 시스템 시연 시나리오

## 🅰️ 시연 1: 소형차 입차 & 자리 변경

1. **소형차 입차**

   - 화면: **[입차 화면]**
   - 차량이 입구 센서를 지나며 `차 번호` 디스플레이에 번호판 출력

2. **첫 번째 추천 (C3)**

   - 푸시 알림: `"C3에 주차하세요"`
   - 화면 이동: [푸시 알림 클릭 → **주차 안내 페이지**]
   - 주차장 지도에서 **C3 자리 강조**, 추천 사유 간단 표시

3. **자리 선점 상황 발생**

   - 실시간 자리 상태 갱신
   - **재추천 (C1)**
   - 푸시 알림: `"C1로 이동하세요"`
   - 화면: [주차 안내 페이지 갱신]
   - 지도상 **C1 자리 하이라이트**, 이동 경로 표시

4. **최종 주차**
   - 화면 자동 이동: [**주차 기록 페이지**]
   - 주차 완료 내역, 추천 이력, 주차 점수 표시

---

## 🅱️ 시연 2: 추천과 다른 사용자 선택

1. **입차 및 추천 (A1)**

   - 푸시 알림: `"A1에 주차하세요"`
   - 알림 클릭 → [**주차 안내 페이지**]
   - 지도에서 A1 안내, 추천 이유 표시

2. **사용자 선택 무시 → 임의 주차 (A3)**

   - 사용자 행동: 추천 무시하고 임의로 **A3**에 주차
   - 센서/영상 판별: 실제 **A3 주차** 자동 감지

3. **시스템 업데이트**
   - 화면 자동 이동: [**주차 기록 페이지**]
   - `"실제 주차 위치: A3"` 내역 및 점수 채점 표시

---

## 🎯 시연 포인트 & 화면 구성

- **추천 알고리즘 시각화**

  - [주차 안내 페이지]에서 추천 자리에 **컬러 하이라이트**
  - **이동 경로 가이드 표시**

- **예외 상황 처리**

  - 자리 선점 시 `"자리 선점됨"` 실시간 안내
  - 추천 자리 변경 시 **푸시 알림 + 화면 즉시 갱신**

- **실사용자 행동 반영**

  - 사용자가 선택한 실제 위치를 **기록 페이지**에 자동 업데이트
  - 추천과 다를 때 **점수 산정 기준** 명확히 표시

- **피드백 제공**
  - [주차 기록 페이지]에서 **주차 성공/실패, 점수, 상세 이력 제공**
  - 추천 위치 vs 실제 위치 비교
  - 사용자 행동에 따른 메시지 노출
