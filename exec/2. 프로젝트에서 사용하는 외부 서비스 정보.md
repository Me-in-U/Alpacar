# 외부 API

## 1. 구글 로그인

로그인을 위한 방법 중 하나로 [Google Identity Authorization OAuth 2.0](https://developers.google.com/identity/protocols/oauth2?hl=ko) 추가

### GOOGLE_CLIENT_ID, GOOGLE_CLIENT_SECRET 생성 가이드

1. [Google API Console](https://console.developers.google.com/?hl=ko) 접속
2. `프로젝트 생성`
3. `API 및 서비스`
4. `사용자 인증 정보`
5. `+ 사용자 인증 정보 만들기`
6. `OAuth 클라이언트 ID`
7. `애플리케이션 유형(웹 애플리케이션)`, `승인된 리디렉션 URI`
8. `만들기`
9. `클라이언트 ID`, `클라이언트 보안 비밀번호` 복사

`Gitlab 소스 클론 이후 빌드 및 배포 가이드.md`의 `4. gitlab-runner 의 홈디렉터리에 .env 파일 생성`에서 `GOOGLE_CLIENT_ID`, `GOOGLE_CLIENT_SECRET`를 발급 `Google API Console`에서 발급 받은 코드로 수정
